## Installation

Install via [VScode](/docs/getting-started/vsc) or copy and paste the potion below into a new file, preferably in a folder called *potions*.

## Potion 

<Potion>
<Fragment slot="contentSlot">
```tsx
import { tv } from "tailwind-variants";

export const avatarPotion = tv({
  slots: {
    root: "h-16 w-16 rounded-full border-2 border-accent",
    fallback:
      "flex h-[inherit] items-center justify-center bg-surface-2 font-bold text-text",
    image: "object-cover",
  },
});

```
</Fragment>
</Potion>

## Components 

### Default

<DemoTabs solidDisabled vueDisabled>
<Fragment slot="react">
```tsx
import {
  AvatarFallback,
  AvatarImage,
  Avatar as AvatarRoot,
} from "@ark-ui/react";
import { avatarPotion } from "@potions/avatarPotion";

const { root, fallback, image } = avatarPotion();

export const Avatar = () => (
  <AvatarRoot className={root()}>
    <AvatarFallback className={fallback()}>ME</AvatarFallback>
    <AvatarImage
      className={image()}
      src="https://i.pravatar.cc/300"
      alt="avatar"
    />
  </AvatarRoot>
);

```
</Fragment>
