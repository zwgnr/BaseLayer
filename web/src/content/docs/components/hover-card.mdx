## Installation

Install via [VScode](/docs/getting-started/vsc) or copy and paste the potion below into a new file, preferably in a folder called *potions*.

## Potion 

<Potion>
<Fragment slot="contentSlot">
```tsx
import { tv } from "tailwind-variants";

export const hoverCardPotion = tv({
  slots: {
    positioner: "rounded-lg bg-surface-3 shadow-xl",
    content: "max-w-96 relative w-96 p-4 text-text-2",
  },
});

```

  Portal,

          />

      </HoverCardTrigger>
      <Portal>
        <HoverCardPositioner className={positioner()}>
          <HoverCardContent className={content()}>
            <HoverCardArrow>
              <HoverCardArrowTip />
            </HoverCardArrow>

              </div>
            </div>
          </HoverCardContent>
        </HoverCardPositioner>
      </Portal>
    </HoverCardRoot>
  );
};
