## Installation

Install via [VScode](/docs/getting-started/vsc) or copy and paste the potion below into a new file, preferably in a folder called *potions*.

## Potion 

<Potion>
<Fragment slot="contentSlot">
```tsx
import { VariantProps, tv } from "tailwind-variants";

export const badgePotion = tv({
  base: "flex items-center justify-center rounded-xl",
  variants: {
    intent: {
      default: "bg-surface-3",
      info: "bg-accent",
      alert: "bg-critical",
    },
    size: {
      sm: "px-2 py-1 text-sm",
      md: "px-4 py-1 text-base",
      lg: "px-6 py-2 text-lg",
    },
  },
  defaultVariants: {
    intent: "default",
    size: "md",
  },
});

export type BadgePotionProps = VariantProps<typeof badgePotion>;

```
</Fragment>
</Potion>
